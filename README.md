The objective of this project is to develop an intelligent Tic-Tac-Toe game in which an AI player competes against a human player.

ðŸ”¹ Project Description

This is a console-based Tic-Tac-Toe game implemented in Python.
In this game:
The human player plays as X
The AI plays as O
The AI evaluates all possible future game states using the Minimax algorithm
Based on this evaluation, the AI selects the best possible move
The Minimax algorithm is a decision-making algorithm commonly used in game theory and artificial intelligence.
